# Install Windows Fonts on macOS

[ğŸ‡¨ğŸ‡³ ä¸­æ–‡](#macos-ä¸‹å®‰è£…-windows-å­—ä½“) | [ğŸ‡ºğŸ‡¸ English](#install-windows-fonts-on-macos)

The built-in Song, Hei, and other fonts on macOS differ from Windows versions in glyph shapes and metrics. The template already bundles Windows-compatible fonts inside the `Font/` directory and loads them automatically during compilation, so **no extra installation is required**. Install them only if you also need these fonts in other macOS apps (Pages, Word, etc.).

## Installation Steps
1. Open the `Font` directory and double-click each of the following font files or drag them all into Font Book:
   - `simsun.ttc`
   - `simhei.ttf`
   - `simkai.ttf`
   - `simfang.ttf`
2. Select "Install" in Font Book. If prompted to overwrite, confirm to ensure consistency with Windows versions.
3. After installation, reopen or compile the thesis project to take effect.

These font packages are for thesis typesetting testing only. Please uninstall all fonts within 24 hours after installation. Do not distribute or use for commercial purposes. Users bear all copyright responsibilities arising from use.
